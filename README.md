# ATM Management System

## Overview

The ATM Management System is a Python-based console application that simulates basic banking operations. It allows users to securely set a PIN, check account balance, deposit money, and withdraw funds through a menu-driven interface.

## Features

* Secure PIN setup and verification
* Balance inquiry
* Cash deposit
* Cash withdrawal
* User-friendly menu-driven system
* Basic transaction management

## Technologies Used

* Python

## Concepts Implemented

* Functions
* Loops
* Conditional Statements
* User Input Handling
* Global Variables

## Future Enhancements

* Transaction History
* PIN Change Option
* File/Database Storage
* Account Number Support
* Exception Handling for Invalid Inputs

## How to Run

1. Install Python.
2. Download the project files.
3. Run:
   python atm.py
